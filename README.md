
#  Full Stack Authentication UI (Bootstrap 5)

##  Project Description
This project is a responsive and visually appealing authentication system built using **HTML, Bootstrap 5, and custom CSS**. It includes multiple pages such as login, registration, forgot password, reset password, and dashboard.

The main goal of this project is to enhance UI/UX design, apply modern styling techniques, and ensure responsiveness across all devices.

---

##  Features
- Responsive design (Mobile, Tablet, Laptop, Desktop)
- Clean and modern UI using Bootstrap 5
- Navigation between all pages
- Styled forms and buttons
- Dashboard layout with navbar
- Custom CSS styling
- Google Fonts integration
- Hover effects and transitions

---

##  Pages Included
- **Login Page (index.html)**
- **Registration Page (register.html)**
- **Forgot Password Page (forgot-password.html)**
- **Reset Password Page (reset-password.html)**
- **Dashboard Page (dashboard.html)**

---

##  Technologies Used
- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript (for page redirection)  
- Google Fonts

1. Download or clone the repository  
2. Open the project folder  
3. Open `index.html` in your browser  
4. Navigate through pages using buttons and links
